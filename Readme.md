# GameJam2017
The game was developed for GameJam 2017 conducted at Collin College. The theme was 'Waves'.

Source code and screenshots can be found here - http://globalgamejam.org/2017/games/wave-me-4

# Description
You play as water molecules cutely named as H2O who has the properties of the wave. Sadly,H2O got lost into the land of sewer and the water don't wave no more. Your job is to help H2O get back into the water and bring back the wave.
